# gpustack-stable-diffusion-3

Stable Diffusion3 GPUStack Custom Backend

Stable Diffusion 3 Async Server Demo

Demo server for testing the stabilityai/stable-diffusion-3.5-medium model.

https://huggingface.co/stabilityai/stable-diffusion-3.5-medium

https://github.com/huggingface/diffusers/tree/main/examples/server-async

## 使用方式

```shell
foyou/gpustack-stable-diffusion-3:latest
python async_server.py --port {{port}} --model_path {{model_name}}
```
